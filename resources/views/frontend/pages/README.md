## الوصف

منصة **تك سوريا** — موقع عربي RTL للمواهب التقنية السورية (Remote-first). Frontend MVP ببيانات تجريبية.

## المميزات

- دليل مواهب + معرض أعمال + ملفات عامة
- وظائف remote — USD — فلاتر client-side
- شركات remote-friendly
- لوحة تحكم للتقني والشركة
- dark mode + localStorage
- Navbar/Footer موحّد عبر `app.js`

## التشغيل

افتح `index.html` في المتصفح أو استخدم Live Server (port 5501).
